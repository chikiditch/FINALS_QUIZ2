Lecture 4: Lists

Ordered Lists

  • Are lists in which each item is numbered usually starting at 1.
  • It is used if you want to emphasize that the elements are ordered or has hierarchy.
  • Are good for set of instructions.

example:

  1. item 1
  2. item 2
  3. item 3
  4. item 4

You also can alter the style of the numbering used for the items with css

example:

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

Unordered Lists

  • Are lists in which each item is not numbered.
  • It is used if you want to emphasize that the elements are unordered or has no hierarchy.
  • Are good for set of instructions.

example:

  • item 1
  • item 2
  • item 3
  • item 4

You also can alter the style of the bullets used for the items with css

example:

  • item 1
  • item 2
  • item 3
  • item 4

Description lists

  • lists in which each item is described by a term.
  • uses dl tag to define the description list, dd defines the term (name), dt describes the term

example:

Notebook
- a small book with blank or ruled pages for writing notes in.
Ballpen
- is a pen that dispenses ink over a metal ball at its point

Nested Lists

  • are lists inside a list, used to organize information

example:

  1. item 1
    • item 1.1
    • item 1.2
  2. item 2
    • item 2.1
    • item 2.2

Horizontal Lists

Target Attributes

When you click a link, the default behavior is to open the link in the same window. You can change this behavior by using the target attribute.

  • _blank - opens the link in a new window or tab.
  • _self - opens the link in the same window or tab (default).
  • _parent - opens the link in the parent frame.
  • _top - opens the link in the full body of the window.
